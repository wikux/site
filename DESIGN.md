## Wikux branding

Almost resembling a sun, Wikux is yellow. Wikux is a clipping of wiki + UX. It is meant to be sleek and professional, having a large focus on quality. Wikux provides free MediaWiki resources on its GitHub profile like MediaWiki extensions, css snippets (future), and widgets.

## Wikux.org site

Wikux.org is a simple website which documents projects and has a few helpful guides for wiki administration with MediaWiki.

The site should follow branding guidelines:

* Acessible on all reasonable devices
* Consistent quality and styling

Notes:

* Wikux yellow: #FFD428 (accent #CDA817)
* Automatic light/dim mode, toggleable
* Sharp edges and calm UI
* Tailwind
* Static only
